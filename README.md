## API Documentation :

### GET /weather/{latitude}/{longitude}
#### Path Variables:
* lat - float64
* lon - float64

#### Sample Response
```
{
  "description": "overcast clouds",
  "temperature": 11.78,
  "weatherCondition": "Clouds",
  "weatherType": "Moderate"
}
```
